PDF Chat
Descripción
PDF Chat es una aplicación innovadora diseñada para interactuar con documentos PDF en lenguaje natural. Permite cargar documentos PDF, realizar consultas en lenguaje natural y obtener respuestas directamente de su contenido. La aplicación utiliza el modelo GPT-3.5-turbo de OpenAI para procesar y entender las consultas.

Características
Carga de documentos PDF: Permite cargar archivos PDF para su procesamiento.

Consultas en lenguaje natural: Realiza consultas en lenguaje natural sobre el contenido de los documentos PDF cargados.

Respuestas generadas por la IA: Utiliza un modelo de inteligencia artificial (IA) para generar respuestas precisas basadas en las consultas proporcionadas.

Interfaz de chat: Toda la interacción se realiza a través de una interfaz de chat intuitiva.

Instrucciones de uso
Clonar el repositorio: Para utilizar la aplicación, primero debe clonar el repositorio de GitHub.


git clone https://github.com/nicoig/ChatPDF.git
Instalar las dependencias: Una vez clonado el repositorio, debes instalar las dependencias necesarias utilizando el archivo requirements.txt proporcionado en el repositorio.


pip install -r requirements.txt
Configurar la API Key de OpenAI: Necesitarás tu propia clave API de OpenAI. Puedes obtenerla creando una cuenta en la página oficial de OpenAI. Una vez obtenida, debes almacenarla en un archivo .env en la misma carpeta donde se encuentra el código de la aplicación.


OPENAI_API_KEY=tu_clave_aqui
Ejecutar la aplicación: Ahora puedes ejecutar la aplicación.


python main.py
Usar la aplicación: Una vez que la aplicación esté ejecutándose, puedes cargar un archivo PDF y comenzar a hacer consultas en lenguaje natural sobre su contenido.
